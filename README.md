# TriviaMag

######This is a game which provokes the user to think fast and test his knowlÐµdge in several areas(for the moment History, Sport, Entertainment, Art and Science). The user is able to challange other users to complete with them. In addition to answering questions the user is able to create questions too, which should be approved by an Admin.
